# ORRER-bot
Me and Alex's shitty discord.py bot

## Requirements:
* [discord.py](https://github.com/Rapptz/discord.py) *(version must be less than v1!)*
* Python 3.4.2+
* `aiohttp` library
* `websockets` library

## Running the bot:
* Make sure to create the file `key.txt` and input your bot token

To run:
`python3 main.py`
